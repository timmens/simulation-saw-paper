## Simulation Code

**A Wavelet Method for Panel Models with Jump Discontinuities in the Parameters**

### Introduction

Here we provide computer code which can be used to reproduce the simulation results in
the paper _A Wavelet Method for Panel Models with Jump Discontinuities in the Parameters_
(Bada O., Kneip A., Liebl D., Gualtieri J. and Sickles R. C.).
